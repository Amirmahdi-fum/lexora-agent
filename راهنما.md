# Lexora 5 — راهنمای نصب و راه‌اندازی

## آپلود در GitHub Pages
همه این فایل‌ها را (همراه پوشه icons) در ریشه ریپازیتوری آپلود کن — دقیقاً مثل قبل:

```
index.html
app.css
app.js
sw.js
lexora.webmanifest
icons/
```

⚠️ نکته: برنامه از تک‌فایل به ۳ فایل (HTML + CSS + JS) تبدیل شده تا نگهداری و آپدیت راحت‌تر باشد. حتماً هر ۳ فایل را با هم آپلود کن.

✅ داده‌های قبلی‌ات (کارت‌ها، تنظیمات) دست‌نخورده می‌مانند — کلیدهای ذخیره‌سازی همان قبلی‌هاست و مهاجرت خودکار انجام می‌شود.

✅ مشکل «آپدیت نمی‌آید» حل شد: Service Worker جدید همیشه نسخه تازه را از GitHub می‌گیرد و اگر نسخه جدیدی منتشر کنی، داخل برنامه پیام «به‌روزرسانی» می‌بینی.

## راه‌اندازی همگام‌سازی (سینک) — یک‌بار برای همیشه
1. در گیت‌هاب برو به: **Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token (classic)**
2. فقط تیک **gist** را بزن (به هیچ دسترسی دیگری نیاز نیست). Expiration را روی No expiration بگذار.
3. توکن را کپی کن → در Lexora: **تنظیمات → همگام‌سازی** → توکن را وارد کن → «اتصال و فعال‌سازی سینک»
4. Lexora خودش یک Gist **خصوصی** در اکانت خودت می‌سازد و همه‌چیز را آنجا نگه می‌دارد.
5. برای دستگاه دوم (مثلاً موبایل): دکمه **«لینک اتصال دستگاه دوم»** را بزن، لینک را برای خودت بفرست (مثلاً Saved Messages تلگرام) و روی موبایل بازش کن. تمام! از این به بعد همه‌چیز خودکار سینک می‌شود.

### چه چیزهایی سینک می‌شود؟
کارت‌ها، پیشرفت و آمار، پروفایل، تنظیمات، دفتر خطاها و تاریخچه گفت‌وگو.
- توکن گیت‌هاب و Gist ID فقط روی دستگاه خودت می‌مانند و هرگز داخل فایل سینک نمی‌روند.
- کلید API فقط وقتی سینک می‌شود که «ذخیره کلید روی این دستگاه» روشن باشد.
- حذف‌ها هم درست سینک می‌شوند (کارتی که در یک دستگاه حذف کنی، در دستگاه دیگر برنمی‌گردد).

## اتصال هوش مصنوعی
تنظیمات → هوش مصنوعی → یکی از دکمه‌های «پرکردن سریع» را بزن (OpenAI, OpenRouter, Groq, Gemini, AvalAI, GapGPT) یا Base URL دلخواهت را دستی بنویس → کلید API خودت → «ذخیره اتصال» → «تست اتصال».

## تلفظ صحیح
- تنظیمات → صدا و ظاهر → **صدای تلفظ**: بهترین صدای انگلیسیِ دستگاهت خودکار انتخاب می‌شود؛ می‌توانی دستی هم انتخاب کنی و «شنیدن نمونه» را بزنی.
- برای کارت‌های جدید، تلفظ IPA و **صدای انسانی واقعی** خودکار از دیکشنری گرفته می‌شود (دکمه ✨ دیکشنری در فرم کارت هم همین کار را دستی می‌کند).
- دیگر هرگز صدای غیرانگلیسی برای متن انگلیسی پخش نمی‌شود.

## نکات نسخه ۵
- چت موبایل تمام‌صفحه شد؛ موقع تایپ، کیبورد دیگر روی کامپوزر نمی‌افتد.
- دکمه ■ برای توقف پاسخ ایجنت وسط جواب.
- «دفتر خطاها» واقعی شد: ایجنت خطاهای پرتکرارت را ثبت می‌کند و می‌توانی مستقیم روی آنها تمرین کنی.
- پاسخ‌های ایجنت به‌صورت زنده (streaming) می‌آیند؛ اگر سرویس تو استریم را پشتیبانی نکند، خودکار به حالت عادی برمی‌گردد.
